# 자료형의 True/False

| 값                    | True / False |
| --------------------- | ------------ |
| "String"              | True         |
| ""                    | False        |
| [1,2,3,4]             | True         |
| []                    | False        |
| ()                    | False        |
| {}                    | False        |
| 1232131 (0이 아닌 수) | True         |
| 0                     | False        |
| None                  | False        |

내용물이 비어있으면 false
