# 파이썬 튜플(tuple)

### 리스트와의 차이점

- ( )으로 둘러싼다

- 정해진 값은 변경불가능

- 1개의 요소만을 가질 때에는 `(1, )`과 같이 ` ,` 을 붙여줘야 함

- `( )` 없이도 쉼표로 구분해 만들 수 있다.

  ```python
  my_tuple = 1,2,3
  ```

  

### 인덱싱

```python
t = (1,2,3,4,5,6)
t[1]
t[4]
```



### 슬라이싱

```python
t1 = (1,2,3,4,5,6)
t1[1:3]
```



### 더하기, 곱하기, 길이구하기

```python
t1 = (1,2,3,4)
t2=('p','q')

#더하기
t1+t2

#곱하기
t1*4

#길이 구하기
len(t1)
```
